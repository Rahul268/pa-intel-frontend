# Deploy to Vercel (Free, 2 minutes)

## Option 1 — Vercel CLI (Recommended)
```bash
npm i -g vercel
vercel login        # Opens browser for auth
vercel --prod       # Deploys from this directory
```
Your URL will appear as: https://pa-intel-xxxxx.vercel.app

## Option 2 — Vercel Dashboard
1. Go to https://vercel.com/new
2. Import Git repository OR click "Deploy from the web" and drag this folder
3. Click "Deploy"

## Option 3 — Netlify
1. Go to https://app.netlify.com/drop
2. Drag the `out/` folder into the page
3. Get your URL instantly (no login required!)
