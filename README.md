# PA Intelligence

Prior Authorization Analytics & Extraction Platform

## Deploy Instantly (Free)

### Netlify Drop (No account needed!)
1. Download `pa-intel-static.zip` from the releases
2. Go to **https://app.netlify.com/drop**
3. Drag the zip file onto the page
4. **Done — get your live URL instantly!**

### Vercel (Free account)
```bash
npm i -g vercel
vercel login
vercel --prod
```

## Screens
- **Dashboard** `/` — Access score analytics, brand charts, submissions table
- **Extraction** `/extraction` — Upload PDFs, extract 12 PA parameters, export CSV

## Extraction API Key
Enter your Anthropic API key in the Extraction screen. Stored in your browser's localStorage only.
